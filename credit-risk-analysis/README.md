# Credit Risk Analysis

## Objective
To analyze customer financial data and identify patterns in loan defaults.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Dataset
Provided as part of UpGrad Data Analytics Program (simulated real-world dataset)

## Key Work
- Data cleaning and preprocessing
- Handling missing values and outliers
- Exploratory Data Analysis (EDA)
- Visualization of key trends

## Outcome
Derived insights to help improve credit risk assessment and customer segmentation.
