# RSVP Movie Analysis (SQL Project)

## Objective
To analyze IMDb movie dataset and provide business recommendations for RSVP Movies.

## Tools Used
- SQL

## Dataset
Provided as part of UpGrad Data Analytics Program (IMDb dataset)

## Key Analysis
- Movie trends by year and month
- Genre performance and ranking
- Actor and director analysis
- Production house insights
- Ratings and vote-based analysis

## Business Insights
- Drama genre identified as most reliable for success
- March found to be optimal release month
- Suggested actors, directors, and production houses
- Recommended multilingual movie strategy

## Outcome
Delivered data-driven strategic recommendations for a successful global movie launch.
